Key Engineering Decisions
  • INTEGER minor-unit balance storage prevents floating-point errors in all financial calculations
  • Sliding-window rate limiting (30 req/min) with append-only audit logs for regulatory traceability
  • Offline HMAC voucher system with anti-double-spend enforcement for low-connectivity markets
  • UUID-based idempotent deduplication ensures zero duplicate transaction processing
  • TTL-cached FX rates reduce external API load while maintaining real-time accuracy
